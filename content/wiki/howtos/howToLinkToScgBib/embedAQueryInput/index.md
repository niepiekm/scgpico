#Embed a query input
You can add a text input field to query the SCG bib anywhere as follows:
```+/scgbib/query+
```
<form method="GET" action="%assets_url%/scgbib"><input type="text" name="query" size="15" maxlength="800"/><input type="submit" value="scg bib"/></form>
